# Characters In Spaces Checklist

Use this checklist to manually verify the milestone demo.

- App opens directly to `Characters In Spaces`.
- Bunk Room, Mess Hall, and Utility Room are visible with connections.
- Mara and Ivan are visible in their starting spaces.
- Selecting a character updates the details panel.
- Movement buttons only show connected destination spaces.
- Moving Mara to Mess Hall updates her location without a page refresh.
- Using Food Station updates Mara hunger and Food Station portions.
- Step runs deterministic behavior and writes action log entries.
- Reset restores the deterministic fixture.
- Save to DB and load from DB restore the current session.
- Save to file and load from file restore the current session.
- Old version 1 alpha/beta/gamma saves are rejected with a clear error.
- Narrow viewport remains usable with wrapped action buttons.
